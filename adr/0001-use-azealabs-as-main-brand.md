# ADR 0001 - Use AZea Labs as the Main Brand

## Status

Accepted

## Context

The ecosystem needs a main brand that can represent multiple products, not only one project.

The brand should support:

* A personal portfolio
* QA Labs
* QA Academy
* Future software projects
* Future mobile or tablet projects
* Technical content and learning resources

The brand also needs to connect with Antonio Zea's personal identity while remaining flexible enough to grow beyond a single personal portfolio.

## Decision

Use **AZea Labs** as the main brand for the ecosystem.

The main domain will be:

```text
azealabs.com
```

## Reasoning

AZea Labs combines personal identity and product flexibility:

* **A / AZ** connects with Antonio Zea.
* **Zea** connects with the personal brand.
* **Labs** communicates experimentation, technology, learning, and software projects.

This name can support different products under one ecosystem.

Examples:

```text
AZea Labs Portfolio
QA Labs
QA Academy
Future Projects
```

## Consequences

### Positive

* The brand is personal but still scalable.
* It can support QA, development, education, and future products.
* It works well with the purchased domain `azealabs.com`.
* It looks professional for a portfolio and future projects.

### Negative

* Some people may not immediately know what "AZea" means.
* The brand may require explanation in the About section.

## Final Decision

AZea Labs will be used as the main ecosystem brand.
