# Azealabs-docs
AZea Labs is a software and QA automation ecosystem created by Antonio Zea.

The goal of this ecosystem is to build professional software projects while improving development, architecture, and automation skills.

## Main Products

- **AZea Labs Portfolio**: personal and professional hub.
- **QA Labs**: platform for practicing web, API, and automation testing.
- **QA Academy**: future educational platform focused on QA Automation.
- **Future Projects**: additional software projects for learning and portfolio growth.

## Purpose

This repository contains the documentation, roadmap, architectural decisions, backlog, and planning documents for the AZea Labs ecosystem.

## Project Philosophy

The goal is not only to build software, but to understand how and why it is built, so the knowledge can later be shared and taught.
